
# pdf_to_markdown.py

## 기능: PDF 파일을 마크다운 형식의 텍스트로 파싱

## 주요 로직: pymupdf4llm 라이브러리를 사용하여 PDF의 텍스트와 레이아웃 구조를 마크다운으로 추출하고 추출된 텍스트를 .md 파일로 저장합니다.

# postprocess_markdown.py

## 기능: 파싱된 마크다운 파일에서 불필요한 행을 제거하고 숫자의 형식을 올바르게 교정

## 주요 로직

### 주요 로직 1: || 제거

- 파싱 과정에서 값이 비어있거나 표 형식이 깨져 ||가 포함된 행을 제거
- 단, 마크다운 표의 헤더와 본문을 나누는 구분선(|---|---|)은 삭제하지 않고 유지

### 주요 로직 2: 소수점 복구

- 문서 파싱 시 소수점이 누락되어 정수로 인식된 데이터를 찾아 원래의 소수점 형식으로 변환
- 3자리 숫자: XXX → X.XX
- 4자리 숫자: XXXX → XX.XX

# 실행 순서 

- pdf_to_markdown.py 실행: 원본 PDF를 마크다운으로 1차 변환
- postprocess_markdown.py 실행: 1차 결과물을 로드하여 데이터 정제 및 최종 파일 생성
